# Arc planning files

Per-arc planning documents for the .NET 10 rewrite. One file per arc, named `ARC-NN-NAME.md`. Each arc file is written **before** the first slice in that arc begins, listing stories, slices, acceptance criteria, and which `BUGS-MITIGATIONS.md` items the arc resolves.

The full arc roadmap and the arc-file template live in `MIGRATION-DOTNET.md`. This README is just an index of arcs as they get created.
